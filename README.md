# ai-agent-commerce-app

AI 에이전트 기반 커머스 애플리케이션

## 기술 스택

- **Framework**: Next.js 15 (Turbopack)
- **Language**: TypeScript
- **UI**: React 19, Tailwind CSS, Radix UI
- **State Management**: Zustand, TanStack Query
- **HTTP Client**: Axios

## 설치 및 실행

### 사전 요구사항

- Node.js 18.17 이상
- npm 또는 yarn

### 설치

```bash
# 저장소 클론
git clone https://github.com/wonhoji-git/ai-agent-commerce-app.git

# 디렉토리 이동
cd ai-agent-commerce-app

# 의존성 설치
npm install
```

### 개발 서버 실행

```bash
npm run dev
```

http://localhost:3000 에서 애플리케이션을 확인할 수 있습니다.

### 프로덕션 빌드

```bash
# 빌드
npm run build

# 프로덕션 서버 실행
npm run start
```

### 기타 명령어

```bash
# 린트 검사
npm run lint
```
